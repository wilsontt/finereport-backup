# FineReport 備份工具

![版本](https://img.shields.io/badge/版本-v1.0.0-blue)
![狀態](https://img.shields.io/badge/狀態-開發中-yellow)

**專案**：FineReport 備份工具  
**說明**：以 Web 方式進行 FineReport 備份，部署於企業入口網站。

---

## 目錄結構

```
4.FineReport備份工具/
├── frontend/          # React + Vite 前端
├── backend/           # Node.js + Express 後端
├── 1.docs/            # 文件
├── Dockerfile.frontend
├── Dockerfile.backend
└── README.md
```

---

## 開發

### 後端

```bash
cd backend
npm install
npm run dev
```

### 前端

```bash
cd frontend
npm install
npm run dev
```
